Introducing xcBuddy for ios

Xcode can interact with ipad by XcodeBuddy plugin.

XcBuddy App running on ios devices is not only an independent source view and analysis tool, but also a client attached to the Xcode. You can connect to xcBuddy app and operate it with XcodeBuddy Plugin.

Current Version

v1.0

Features

  • File tree——“Similar to Project Navigator of Xcode” , and will provide more source code analysis functions.
  • Source code syntax highlighting, and will support more languages.
  • Operate ios device with Xcode,the Gospel of extreme fast keyboarder,and will suppport more operations.

Requirements

  1. XcodeBuddyPlugin for Xcode,installation method.
  2. xcBuddy for ios,running on ios platform.

Way Of Connection

Wi-Fi

Supported language

objective-c、objective-c++、swift、C、C++

Code Transmission

Two methods:

Using XcodeBuddy Plugin

  1. Ensure the connection of xcBuddy app and XcodeBuddy plugin.
  2. In xcode,click the menu of Edit->XcodeBuddy->Send All Project Files，and click Send button.

USB Data Line

  1. Connect the computer to the ipad with USB data line.
  2. Run iTunes, and select the tab of Apps.
  3. Select xcBuddy in the Apps list of File Sharing,drag and drop the code directory into the right list,or click the Add button bottom.

Source Files Tree

Add Projects

  1. click + button in xcBuddy app, and projects list pops up from the bottom.
  2. select projects what you want to add from the list,and click the Done button.

Delete Project Reference

Long press on the project icon you want to delete,and click the Delete .
Tips: Click here to remove the project reference.You cannot remove the project files.

Delete Project Files

  1. Click + button in xcBuddy app, and projects list pops up from the bottom.
  2. Select projects what you want to delete from the list, and click the delete button.

Interaction between ‘xcBuddy’ app and ‘XcodeBuddy’ plugin

The most of interaction functions is showed up in the ‘Edit’ menu of Xcode ,as follows:

  • Connect
    The status bar show the IP address of the ios device in your LAN at the bottom of ‘xcBuddy’ app,as follows:

    Click the Connect menu item in the XcodeBuddy menu of Xcode,will show:

    input the IP address and port number of your ios device, and click the Connect button.The connected icon will change to on the xcBuddy app.
    Tip: The ‘XcodeBuddy’ menu shows up IP addresses of the most-recently connected ios devices.

  • Send All Project Files
    Send all the project files to ‘xcBuddy’ app.

  • Disconnect
    Disconnect ‘xcBuddy’ app from the network.
  • Help
    Show the version number and the installation help of plugin.
  • Open With xcBuddy

    Right click on the source file in the ‘Project Navigator’ of Xcode, and locate the menu item named ‘Open With xcBuddy’ in the right mouse context menu.Click it, and the file will be sended to and show in ‘xcBuddy’ app.

Switch Of Locking Screen

Click this button in the top right corner, and click the menu item named ‘Lock Screen’.It can shut down or open up the system function of locking screen automatically.

Next version

v1.2

Will Provide

  • can download and use the code theme files of Xcode in ‘xcBuddy’ app.
  • can config fonts and font-size.
  • can use groups of Xcode in ‘xcBuddy’ app.
  • more perfect code highlighting.
  • will provide the function of feedback.
